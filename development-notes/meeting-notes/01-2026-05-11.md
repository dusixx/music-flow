## Что обсуждали

- Знакомство команды и ментора с командой
- Обсуждение стека разработки (Angular 21 + Angular Material)
- Backend: ориентир Firebase
- Обсуждение и утверждение структуры проекта
- Согласование правил создания и review для Pull Request (далее PR)

- Рекомендации ментора Алексея по изучению Angular

## Что решили

- Приложение будет написано на актуальной версии Angular с использованием последней версии Angular Material
- Для будущей авторизации будет использоваться Firebase
- Использовать Figma Design как опору для разработки UI
- Создавая PR мы будем ожидать обязательный approve от ментора
- Никакие коммиты не сквошить.
- Каждый PR будет содержать ссылку на issue
- Необходимо изучить работу Dependencies Injection, Strategy и changeDetection
- Созвоны будут устраиваться еженедельно с предварительным утверждением времени
- Ментор Алексей будет приглашен на созвоны в случаях необходимости по предварительному согласованию
- Длительность первого митинга: примерно 2 часа, формат оценён как продуктивный
